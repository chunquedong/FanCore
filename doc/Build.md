


### Build from Sources ###
```
  cd src
  sh build_all.sh
  sh test_all.sh
```

### Build More Ext Librarys ###
clone hg repo
```
  hg clone http://bitbucket.org/chunquedong/fan-1.0
```
build:
```
  cd src
  sh build_lib.sh
```

### API Docs ###

```
  fan compilerDoc -all
```
